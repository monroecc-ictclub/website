# Monroe Community College Information & Computer Technology Club Website

This repository contains the official website for the Monroe Community College Information & Computer Technology Club.

The website is maintained by club members and faculty as a collaborative learning project. In addition to promoting club activities and events, the repository provides students with hands-on experience using Git, GitHub, version control, and collaborative software development practices.
