Upcoming Events

The ICT Club hosts meetings, workshops, guest speakers, and hands-on activities throughout the semester. Our goal is to help students develop technical skills, collaborate on projects, and connect with others who share an interest in computing and technology. The schedule below lists our planned activities for the Fall 2026 semester. Click on any event to view additional information.

Aug. 31, 2026 (Mon)
ICT Meet & Greet: ICT Department and ICT Club Meeting

The Information & Computer Technologies Department and the ICT Club invites you to attend our annual ICT Meet & Greet event. This is a great opportunity to learn more about our academic programs, meet faculty and staff, connect with fellow students, and discover how you can become involved with the ICT Club.

What You'll Do

  • Learn about ICT programs and courses
  • Meet ICT faculty and staff
  • Learn about the ICT Club and student opportunities
  • Connect with fellow students
  • Enjoy free pizza

Date: Monday, August 31, 2026

Time: 12:00 PM – 1:00 PM

Location: Room 09-263

Skill Level: Everyone

Recommended Prerequisite: None

Sep. 09, 2026 (Wed)
GitHub 101: Introduction to Version Control

The ICT Club invites you to attend our Git & GitHub 101: Introduction to Version Control workshop. Version control has become an essential skill for software developers and is now widely used throughout industry and open-source software projects. This introductory workshop provides a hands-on introduction to Git and GitHub while explaining how version control helps individuals and teams manage software projects more effectively

What You'll Do

  • What is Version Control Software (VCS)?
  • Why version control is important
  • What is Git?
  • What is GitHub?
  • How Git and GitHub work together
  • Using Git in VS Code and CCS

Date: Monday, August 31, 2026

Time: 12:00 PM – 1:00 PM

Location: Room 09-263

Skill Level: Everyone

Recommended Prerequisite: None This workshop is especially recommended for students enrolled in CPT210 and CSC202.
