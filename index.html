About the Club

The Information & Computer Technology Club brings together Monroe Community College students who are interested in computing and technology. Whether you're studying Computer Science, Computer Engineering Systems, Information Technology, or simply enjoy learning about technology, the club provides opportunities to explore programming, cybersecurity, networking, PC hardware, diagnostics, Linux, web development, and other emerging technologies through workshops, projects, competitions, and collaboration with fellow students.

Topics

Programming

Learn programming fundamentals using languages such as C, C++, Python, Java, and JavaScript while building practical software projects.

Cybersecurity

Explore computer security through hands-on activities covering ethical hacking, system security, and cybersecurity best practices.

Networking

Learn how computers communicate through wired and wireless networks while exploring routing, switching, and network services.

PC Build & Repair

Build, upgrade, troubleshoot, and maintain desktop computers while learning about modern computer hardware.

Diagnostics

Develop troubleshooting skills using multimeters, oscilloscopes, logic analyzers, soldering equipment, and other diagnostic tools.

Linux

Explore Linux command-line tools, shell scripting, system administration, networking, and open-source software.

Web Development

Design and develop modern websites using HTML, CSS, JavaScript, and related web technologies.
