# Contributing to the ICT Club Website

Thank you for your interest in contributing to the Monroe Community College Information & Computer Technology Club website.

This website is maintained by students and faculty as a collaborative learning project. Whether you are new to GitHub or an experienced developer, your contributions are welcome.

## Getting Started

1. Clone the repository.
2. Create a new branch for your changes.
3. Make small, focused improvements.
4. Test your changes locally before committing.
5. Commit your work with a clear, descriptive message.
6. Submit a Pull Request for review.

## Development Guidelines

* Keep HTML semantic and easy to read.
* Keep CSS organized and well commented.
* Use meaningful names for files, classes, and IDs.
* Avoid unnecessary complexity.
* Write code that another student can easily understand.

## Commit Messages

Use short, descriptive commit messages.

Examples:

* Add Topics cards
* Update hero section
* Improve mobile layout
* Fix navigation spacing
* Update Upcoming Events section

## Coding Style

* Use consistent indentation throughout the project.
* Place related code together.
* Add comments to explain non-obvious code.
* Keep lines reasonably short for readability.
* Test the website after every significant change.

## Pull Requests

Before submitting a Pull Request, verify that:

* The website loads correctly.
* Links function as expected.
* The layout works on both desktop and mobile devices.
* Existing functionality has not been broken.

## Project Goals

This website serves two purposes:

1. Promote the Information & Computer Technology Club.
2. Provide students with experience using professional software development practices, including Git, GitHub, version control, code reviews, and collaborative development.

## Questions

If you have questions about contributing, please visit the Contact page on the club website. You can also reach out to one of the current club officers or the faculty advisor using the contact information provided there.
